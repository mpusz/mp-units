# Example Tags

Browse **mp-units** examples by tags to quickly find examples relevant to your interests.
Examples are organized by complexity level, unit system, library features, and application
domain.

## Finding Examples

All examples are tagged by complexity level, unit system, library features, and application
domain. Use the interactive tag index below to filter examples by any combination of tags.

### By Level

Filter examples by your experience level:

- **Level - Beginner** - Detailed explanations of fundamentals
- **Level - Intermediate** - Feature-focused demonstrations  
- **Level - Advanced** - Complex, real-world applications

### By System

Find examples using specific unit systems (SI, CGS, Imperial, Natural, IAU, and more).

### By Feature

Discover examples demonstrating particular library capabilities (conversions, affine space,
dimensionless quantities, custom units, generic programming, mathematical operations, and more).

### By Domain

Locate examples relevant to your field (physics, electronics, aerospace, embedded systems,
navigation, and more).
