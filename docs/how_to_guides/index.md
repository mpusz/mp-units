# How-to Guides

Welcome to **mp-units** How-to Guides! These task-focused guides help you solve specific
problems and accomplish particular goals when working with the library.

## What are How-to Guides?

How-to Guides are **task-oriented** instructions that show you how to solve specific
problems and accomplish particular goals.

Each guide:

- ✅ Addresses a specific problem or use case
- ✅ Provides practical, actionable steps
- ✅ Assumes familiarity with library basics
- ✅ Focuses on getting things done

**Prerequisites**: Complete the [Tutorials](../tutorials/index.md) first to learn the
fundamentals, then refer to these guides when you need to solve specific problems.

## When to Use How-to Guides?

Use How-to Guides when you:

- Need to solve a specific problem
- Want to integrate **mp-units** with existing code
- Are migrating from another library
- Need to customize library behavior for your use case
- Want to maximize compatibility or interoperability
